Survive Tetris | C++

An action game based on the classic puzzler.

About the Game

Survive Tetris is a 2D action game programmed in C++ using the SDL graphics library. Tetris pieces fall randomly from the top of the screen, and players try to escape from the level. The game is highly customizable, and players can change the width, height, and cell size of the game screen, as well as the difficulty (game speed).

My Contributions

  • Programming (MVC pattern, C++, SDL)
  • Helped design the game model and controller
  • Troubleshooting and tweaking the game view

The Process

We spent a long time discussing the best way to represent the Tetris board and pieces before we started programming. We based our game structure on the Model-View-Controller pattern we learned in our object-oriented design class. We first developed an initial grid and tetromino class, then implemented random block generation and collision. We created a custom Square class to improve performance and allow easy board scaling.
